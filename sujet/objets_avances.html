Quelques objets plus complexes

Povray offre de nombreuses autres primitives géométriques, permettant la modélisation et le rendu d'objets beaucoup plus complexes. Dans cette partie, vous allez en étudier quelques unes ...

Les objets de révolution

Un objet de révolution est défini par une courbe 2D et un axe de révolution (ou de rotation) : l'objet est obtenu comme la surface résultant de la rotation de la courbe autour de son axe.
Povray dispose de deux types d'objets de révolution : les lathes et les sor. L'exemple ci-dessous donne un premier aperçu de l'utilisation des lathes :
#include "colors.inc"

camera {location <0,7,-10> look_at <0,0,0> }
background { White }
light_source { <100,100,-200> rgb 0.9}

lathe {
  linear_spline
  5,
  < 3, 0 >, < 4, 0 >, < 5, 5 >, < 3, 5 >, < 3, 0 >
  pigment { YellowGreen }
  finish {ambient .3 phong 1}
}

Application

1. Modifiez le code précédent de manière à obtenir cette forme.
2. Modifiez de nouveau ce code pour obtenir cette nouvelle forme.
3. Testez à présent les splines quadratiques et cubiques sur la forme de la question précédente. 
4. Recherchez les modifications très simples à apporter à la forme initiale
(en termes de point(s) supplémentaire(s) et de spline) pour obtenir la forme ci-contre.
Il est conseillé de commencer par obtenir une forme générale à l'aide des splines linéaires.

Les "Height Fields"

Aussi appelés Modèles Numériques de terrain (MNT), les Height Fields sont principalement utilisés pour modéliser du relief, que ce soit pour des terrains (montagnes) ou pour des bas-reliefs.
