les motifs fractals

Ces motifs utilisent des algorithmes fractals bien connus. Afin de bien comprendre la portée de certains de leurs paramètres, voyez la page sur les fractales dans la section des concepts généraux.

mandelbrot

Ce motif représente le célèbre ensemble de Benoît Mandelbrot avec les options suivantes :

mandel NbIterations  // Entier (entre 2 et 33)
 (exponent Exposant) // Entier
 (interior Type,Facteur) // Entier , Décimal
 (exterior Type,Facteur) // Entier , Décimal

Le nom du motif est suivi du nombre d'itérations, dont dépend la 'précision' du résultat.

L'option exponent permet de spécifier la valeur de l'exposant dans la formule Z = Zx + C. Plus cet exposant est élevé, plus le calcul est long, surtout au-delà de 4. La valeur par défaut est 2.

Les optionsinterior et exterior contrôlent la façon dont les valeurs des zones du motif seront réparties. La partie intérieure, c'est l'ensemble des points pour lesquels la formule converge. C'est l'îlot 'cardioïde' au centre du motif.

L'extérieur, c'est tout ce qu'il y a autour (c'est à dire les contours successifs pour les niveaux d'itérations inférieurs à la valeure maximale).

Le type d'interior ou exterior a la signification suivante :
(les illustrations modifient uniquement exterior)
0 : Prend la valeur 1

1 : Pour exterior, la valeur dépendra du nombre d'itération avant que le seuil de convergence/divergence soit atteint. Cette valeur ne sera pas affectée par le facteur.
1 : Pour interior, la valeur dépendra de la valeur absolue du plus petit point dans l'orbite du point calculée (euuuuh...).

2 : La valeur sera la partie réelle du dernier point dans l'orbite.

3 : La valeur sera la partie imaginaire du dernier point dans l'orbite.

4 : La valeur sera le carré de la partie réelle du dernier point dans l'orbite.

5 : La valeur sera le carré de la partie imaginaire du dernier point dans l'orbite.

6 : La valeur sera la valeur absolue du dernier point dans l'orbite.

Le facteur qui suit multiplie le résultat avant de renvoyer la valeur définitive du motif en un point.

Par défaut, interior est réglé à 0 et extérior est réglé à 1.

julia

La fractale de Gaston Julia est un sous-ensemble de celle de Mandelbrot. En fait, c'est l'ensemble de Mandelbrot qui est le 'catalogue' de tous les ensembles de Julia possibles : à chaque point du Mandelbrot correspond un ensemble de Julia ! Voici la syntaxe :

julia <Complexe> //Nombre complexe
      NbIterations     // Entier
 (exponent Exposant) // Entier
 (interior Type,Facteur) // Entier , Décimal
 (exterior Type,Facteur) // Entier , Décimal

Le nombre complexe donne la position de l'ensemble dans le plan des nombres complexe. Il s'exprime sous la forme d'un vecteur à deux composants, la partie réelle et la partie imaginaire. L'illustration ci-dessus utilise le nombre complexe <.5,.1>.

magnet

Une autre variante 'magnétique' des motifs fractals. Syntaxe :

magnet 
 TypeMagnet
 mandel / julia <Complexe> 
 NbIterations     // Entier
 (exponent Exposant) // Entier
 (interior Type,Facteur) // Entier , Décimal
 (exterior Type,Facteur) // Entier , Décimal

TypeMagnet peut prendre la valeur 1 ou 2, et il faut choisir entre les algorithmes julia et mandel.
