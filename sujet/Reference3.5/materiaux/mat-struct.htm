structure des matériaux

Le système de matériaux de POV-Ray est basé sur un ensemble de "composants" assemblés dans une structure logique. Chacun de ces composants définit une caractéristique particulière des matériaux : couleurs, aspect de surface, etc...

Couleur : une couleur, opaque ou transparente.
Motif : répétition spatiale, suivant une forme, une régularité,... d'un ensemble de couleurs ou d'autres choses.

Pigment : répartition de la ou des couleurs d'une surface.
Finish : caractéristiques de "finition" d'une surface, réflexion, rehauts,...
Normal : apparences de grain, relief,... d'une surface.

Interior : caractéristiques de l'intérieur des objets, réfraction, densité,...
Media : densité de "particules" à l'intérieur des objets, flammes, poussière,...

XX_map : palette constituée d'éléments de même nature (couleurs, pigments,...).

Texture : structure regroupant les caractéristiques de la surface d'un matériau (pigment + finish + normal) ou une combinaison de sous-structures de texture.

Material : structure regroupant, pour la commodité de la syntaxe, texture et interior (sinon, il serait impossible de créer une définition de verre coloré sous un seul nom, par exemple).

pigment, finish, normal et interior peuvent être utilisés isolément, en tout ou en partie :

object {
 MonObjet
  pigment {Red filter .5}
  normal {bumps .5}
  interior {ior 1.5}
}

...ou bien s'inscrire dans une structure plus générale ou plus complète :

object {
 MonObjet
 material {
  texture {
   pigment {...}
   finish {...}
	 }
  interior {...
	 media {...}
	}
 }
}

Tous les composants peuvent être déclarés :

#declare MaCouleur = color rgb <1,1,0>;

#declare MonPigment = pigment {MaCouleur filter .5}

#declare MaTexture = 
 texture {
   pigment {MonPigment}
	 finish {phong .7}
 }
