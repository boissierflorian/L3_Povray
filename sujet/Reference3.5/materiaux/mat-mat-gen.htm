vers l'index général de l'aide - index des matériaux

material

material est une structure qui permet tout simplement de regrouper une définition de texture et une définition d'interior, afin de pouvoir les associer à un seul nom de variable.

En effet, lorsque POV-Ray 3.1 a transféré la réfraction de finish vers interior, pour plus de cohérence, il aurait été pénible de devoir spécifier une texture de verre de cette manière pour chaque objet :

object {
 MonObjet1
 texture {Bleu_Translucide}
 interior {ior 1.5}
}

Grâce à material, il suffit de faire ceci :

#declare MonVerre =
 material {
 texture {Bleu_Translucide}
 interior {ior 1.5}
}

object {
 MonObjet1
 material {MonVerre}
}
