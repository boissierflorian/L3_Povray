ambient_light

objectif

Régler le niveau de la lumière ambiante d'une scène.

compatibilité

POV-Ray 2.0 +sup.

syntaxe

global_settings {
        ambient_light   //couleur
       }

La lumière ambiante est utilisée par POV-Ray pour simuler grossièrement l'effet de réflexion diffuse, qui est responsable de l'éclairage des zones qui sont partiellement ou complètement dans l'ombre. ambient_light est là pour modifier facilement l'intensité de l'éclairage ambiant sans avoir à modifier toutes les déclarations "ambient" qui se trouvent dans les "finish" de toutes les textures utilisées dans la scène (y compris les finish par défaut...). Cela permet aussi d'obtenir des effets intéressants, par exemple en modifiant la couleur de la lumière ambiante.

La valeur par défaut de la lumière ambiante est rgb <1,1,1>. On peut spécifier une couleur avec un canal filter ou transmit (rgbft <.6,.8,.5,.6,.2>, par exemple), mais seules les composantes rgb sont prises en compte. La valeur d'ambient réellement utilisée lors du rendu est celle-ci:

ambient = (ambient du finish)*(ambient_light)

exemple

 
//pour assombrir la scène
global_settings { ambient_light rgb .8 }

//pour éclaircir la scène
global_settings { ambient_light rgb 1.2 }

//pour donner un aspect bleuté aux ombres
global_settings { ambient_light rgb <.9, .9, 1.2> }

Remarque

ambient_light est un paramètre influent lorsqu'on utilise la radiance. La radiance étant destinée elle-aussi à modéliser (de manière plus précise) l'effet de la réflexion diffuse, on aura tendance à minimiser l'effet de la lumière ambiante à l'aide de ambient_light.
