global_settings - paramètres d'une scène

global_settings est un mot clé fourre-tout, qui permet de regrouper un certain nombre de paramètres de la scène. Vous pouvez le déclarer n'importe où dans la scène, à l'exclusion de l'intérieur d'une autre déclaration (c'est-à-dire en dehors de tout objet, caméra, source lumineuse, etc...). Vous pouvez aussi l'utiliser plusieurs fois en configurant des paramètres différents à chaque fois. Quoi que vous déclariez pour un paramètre donné, ce sera toujours la dernière déclaration (dans l'ordre d'apparition dans la scène) qui sera prise en compte, et ceci pour la scène entière. Si vous déclarez number_of_waves 5 pour un premier motif ripples, puis vous déclarez number_of_waves 10 juste avant un second motif ripples, les deux motifs utiliseront number_of_waves 10 (puisque c'est la dernière déclaration de ce paramètre dans le fichier).

Exemple

global_settings {
	max_trace_level 50  //prévoir énormément de réflexions, réfractions et cie,
                            //aïe aïe aïe le temps de calcul...
	assumed_gamma  2.1  // régler gamma
	ambient_light  1.5  // éclaircir la scène
}

....

global_settings {
	ambient_light  0.8  // finalement j'ai changé d'avis, j'assombris la scène...
}
