charset

objectif

Indiquer quel jeu de caractères est employé pour les chaînes.

compatibilité

POV-Ray 3.5

syntaxe

global_settings {
        charset ascii / utf8 / sys
        }

Le jeu de caractères choisi sera utilisé pour TOUTES les chaînes spécifiées dans la scène, y compris les noms de fichiers !

ascii est le jeu de caractères par défaut, mais il accepte uniquement les caractères ASCII standard de 0 à 127.

utf8 spécifie l'emploi du jeu de caractères unicode. C'est le seul moyen de placer des caractères spéciaux dans vos objets texte : accents, symboles, lettres grècques,...

sys n'est pas actuellement implémenté.

remarque

Un caractère unicode se place dans une chaîne comme ceci : \uNNNN, NNNN étant le numero à quatre chiffres (en hexadécimal) que vous pouvez trouver dans les tables unicode. Exemple pour "sphère" :

#declare MaChaine = "sph\u00E8re"
