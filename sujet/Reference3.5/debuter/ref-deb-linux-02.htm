prise en main de POV-Ray pour Linux - installation simple

On considère ici que vous ne voulez pas vous amuser à recompiler POV-Ray pour Linux et que vous souhaitez faire une installation de POV-Ray accessible à tous les utilisateurs de la machine, et que vous avez les droits "root" (administrateur).
Mefiance avec les privilèges root, c'est le moyen idéale de faire des dégats...

En principe, l'installation est aisée, accessible a tous avec juste quelques notions de système Unix. La première chose à faire, c'est de décompresser l'archive dans le répertoire désiré :
/usr/local/lib
est un bon choix (c'est là que l'on place généralement les applications non installées par la distribution), mais vous pouvez l'installer où bon vous semble.

Passez en mode administrateur (root) afin d'avoir le droit de procéder à l'installation : 

zorro$ su
Password: ...

Passez dans le répertoire d'installation et lancez la commande de décompression : 

root$ cd /usr/local/lib
root$ tar zxvf /home/zorro/povlinux.tgz

Vous devriez voir une liste de fichiers qui commence et termine comme ceci :

povray-3.5/
povray-3.5/include/
povray-3.5/include/ash.map
[...]
povray-3.5/povray-3.5/gamma.gif.txt
povray-3.5/povray-3.5/README
povray-3.5/povray-3.5/README.unix

Cette commande a du créer un repertoire povray-3.5/. vous pouvez le verifier en tapant: 

root$ ll pov* -d
...
drwxr-xr-x    5 root      root          4096 déc  7 16:15 povray-35/
...

Si les droits (drwxr-xr-x) ou les propriétaires ( root    root) ne sont pas les bons il faut les changer avec :

root$ chown root.root -R ./povray-35/
root$ chmod 755 povray-35/

POV-Ray est maintenant installé sommairement. Placez-vous dans le répertoire d'installation de POV :

cd povray31/

Faisons un petit essai en lançant POV-Ray avec quelques options :

root$ cd povray-35/
root$ ./povray -I./scenes/advanced/skyvase.pov -L./include/ +V +D -w320 -h240 +FT +Ovase.tga

... ce qui devrait lancer le rendu de la scène Skyvase
