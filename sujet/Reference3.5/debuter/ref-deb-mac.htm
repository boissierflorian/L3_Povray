prise en main rapide de POV-Ray 3.5 pour Mac OS

Logiciels nécessaires :

StuffIt Expander[TM] 6.0.1 de Aladdin Software.

Permet de décompresser les fichiers téléchargés. StuffIt Expander permet aussi de monter des images créées par Shrinkwrap ou Apple Disk Copy.

Pour télécharger une copie GRATUITE de Expander, visitez le site web de Aladdin.
L'extension "CarbonLib" est nécessaire pour utiliser POV-Ray 3.5. Sous MacOS 8.1, il faut au minimum la version 1.0.4, sous MacOS 8.6 et supérieur, il faut la version 1.6 au minimum.

Cette extension est téléchargeable à partir du site d'Apple, dans la section "Mise à jour logiciels".

Téléchargement :

POV-Ray 3.50 se télécharge à partir du site POV-Ray.
Une fois le téléchargement effectué, vous trouver sur votre disque dur le fichier : POV_RAY_35.IMG.

Suivant la configuration de votre Mac, vous pouvez soit double-cliquer sur cette icône, soit déplacer celle-ci sur l'icône de StuffIt Expander. Un disque virtuel apparaît alors sur le bureau.

Installation :

Pour installer POV-Ray, il suffit de déplacer l'icône du disque virtuel vers le disque dur ou dans un sous-dossier. Un nouveau dossier portant le même nom que le disque virtuel est créé. Nous pouvons maintenant démonter le disque virtuel en déplaçant celui-ci vers la Corbeille.

Si nous ouvrons ce dossier, cette image doit apparaître :

Nous devons apporter une petite modification à cette installation :
-Supprimer le programme "POV-Ray Mac 3.5"
-Ouvrir le dossier "POV-Ray Mac 3.5 folder"

-Créer un alias de "POV-Ray Mac 3.5" (Sélectionner le fichier puis appuyer sur COMMAND-M).
-Déplacer cet alias dans le dossier supérieur, sur le bureau, dans le dossier "Menu Pomme",... en fait ou vous voulez ! C'est cette icône qui servira à lancer POV-Ray.

Première image :

Une fois le programme lancé, nous allons créer notre première image. Dans le menu "File", nous choisissons "New". Dans la fenêtre qui apparaît, nous tapons le texte suivant : 

#include "colors.inc"

camera { location <40,50,36> look_at <0,15,0> }

plane {
	y,0 
	pigment { checker color OrangeRed color YellowGreen scale 10 }
	rotate y*-30
	}

sphere {
	<0,19,0>,19
	pigment { SteelBlue filter 0.5 transmit 0.2 } 
	finish { reflection 0.4 phong 0.7 }
	}
        
light_source { <45,80,125> White*2 }

fog { White distance 550 }

Nous enregistrons sous le nom "essai". Dans le menu "Edit", nous choisissons "image Setting". Dans l'onglet "Output" nous sélectionnons :
-Image Size : 320x240
-Save image as : PICT (MacOS)
Dans le menu "Render", nous choisissons "Render essai Setting".
