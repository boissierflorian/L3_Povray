Géométrie Constructive des Solides (CSG, Constructive Solid Geometry)

objectif

Grouper et combiner des solides.

compatibilité

POV-Ray 2.0+sup

explication

Un solide est un objet dont l'intérieur et l'extérieur sont bien définis.

La CSG permet de modifier des volumes solides. On peut, grâce aux fonctions intégrées : couper un objet avec un autre, faire une union de 2 objets (ou plus), créer un nouveau volume par l'intersection d'objets différents.

union et merge

union permet l'union de plusieurs volumes pour obtenir un seul objet. Voici sa syntaxe :

union {
	Objet1
	Objet2
	Objet3
 ...
 ObjetN	
 (...modificateurs...)
       }

merge (fusion) permet, comme union, d'assembler plusieurs volumes en un seul mais ne laisse pas apparaître les contours intérieurs (utile pour les objets transparents ou semi-transparents).

difference

difference permet d'obtenir un nouvel objet par soustraction de volume(s). Voici sa syntaxe :

difference {
	Objet1 //la forme de laquelle on va retrancher des parties.

	Objet2 //la première forme de soustraction
	Objet3 //la deuxième...
	...
 ObjetN	

 (...modificateurs...)
}

intersection

intersection permet d'obtenir un nouvel objet dans lequel seules les parties communes sont visibles. Sa syntaxe est semblable à celle de union.

En interne, POV-Ray traite la fonction difference comme une intersection dont le premier objet voit son intérieur/extérieur inversés.

 difference {
  object {Objet1}
  object {Objet2}
}

...est parfaitement équivalent à...

 intersection {
  object {Objet1}
  object {Objet2 inverse}
}
