camera

objectif(!)

Définir la position de l'oeil de l'observateur dans la scène et le type de projection (perspective) utilisée.

compatibilité

POV-Ray 2.0 +sup. pour le premier groupe de fonctions.
POV-Ray 3.0 +sup. pour les fonctions de flou, perspectives, et normale.
POV-Ray 3.5 +sup. pour la projections sphérique.

syntaxe générale


 camera{
         /* modes de projection */
        (perspective / orthographic / fisheye / 
         ultra_wide_angle / omnimax / panoramic / 
         cylinder TypeCylindre / 
         spherical)
         /* positionnement */
        location PointDeVue          //vecteur
        (right VecteurLargeur)       //vecteur
        (up VecteurHauteur)          //vecteur
        (direction VecteurDirection) //vecteur
        (sky VecteurCiel)            //vecteur
        (angle Degrés,(Degrés))      //nombres décimaaux
        look_at PointRegardé         //vecteur
        
         /* profondeur de champ */
        (blur_samples Nbre_Echantill_Flou)  //nombre entier
        (aperture Grandeur)                 //nombre décimal
        (focal_point PointFocal)            //vecteur
        (confidence ConfianceFlou)          //nombre décimal
        (varience SeuilDeVariation)         //nombre décimal

         /* effets */
        (normal {...})
         
        (...transformations spatiales...)
        }

Toutes ces fonctionnalités sont détaillées dans les pages suivantes.

page suivante : positionnement de la caméra

exemple

 
camera {location <15,24,25> look_at <1,4,0> angle 50}

remarques

L'ordre des commandes de la caméra n'a pas d'importance, sauf pour le mode de projection qui doit impérativement être spécifié en premier lieu.

On peut bien entendu déclarer une caméra comme tout autre objet :

#declare Cam1 = camera {...}

C'est utile lorsqu'on veut changer rapidement de point de vue pendant qu'on travaille sur une scène.
