fonctions 3D pour les motifs et les isosurfaces

POV-Ray propose un nombre important de fonctions 3D utiles pour les isosurfaces ou les motifs. Toutes ces fonctions sont codées 'en interne' dans POV-Ray, il y en a actuellement 78. En principe on pourrait appeler ces fonctions comme ceci :

function {internal(N)}

N étant un nombre entre 0 et 78. Mais cette méthode est déconseillée. D'abord, parce que, dans une future version, la signification d'un numero pourrait changer ! Ensuite, ce n'est pas très explicite.

Le fichier include standard "functions.inc" redéfinit toutes ces fonctions, en leur donnant un nom explicite. La plupart de ces fonctions doivent être suivies de plusieurs paramètres, x, y, z et d'autres paramètres plus spécifiques.

Seule une partie de ces fonctions sont documentées précisément sur cette page. Les documenter toutes représenterait un travail colossal et, finalement, peu utile. Nous avons donc sélectionné les plus intéressantes d'entre elles (mais elles sont toutes mentionnées).
