Les fonctions Povray

Le langage de script de Povray vous autorise à créer des fonctions (ou plus exactement des macros), proches de ce qui existe dans les langages de programmation traditionnels.

Application

Ecrire la macro de prototype suivant : rondelle(diam_ext, diam_int, epaisseur) qui crée une rondelle (un cylindre creux) avec la signification suivante pour les paramètres :
  • diam_ext le diamètre extérieur de la rondelle ;
  • diam_int son diamètre interieur ;
  • epaisseur son épaisseur.
Par défaut, la rondelle sera créée dans le plan Oxz, centrée en O et posée sur le plan Oxz. La couleur (ou la texture) ne sera pécisée qu'au moment de l'appel de la macro. A titre d'exemple, l'exécution du code suivant donnera la figure en regard :
object {
  rondelle(2,1.5, 1.0)    
  pigment {Red}
}

object {
  rondelle(3, 2, 0.2)
  pigment {Green}
  translate <-2, 3, 0>
  rotate<0,0,40>
}
