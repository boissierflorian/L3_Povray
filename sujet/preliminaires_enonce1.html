Préliminaires

POV-Ray ou Persistence Of Vision Raytracer est un logiciel gratuit qui permet de réaliser des images de synthèses. Bien que gratuit POV-Ray n'est pas un Logiciel libre (au sens de la licence GPL): il a été créé avant l'apparition de la notion de Logiciel libre.

POV-Ray crée des images tridimensionnelles photo-réalistes en utilisant le tracé de rayons. Il lit dans un fichier texte -- que nous appellerons script dans toute la suite -- les informations décrivant les objets et les sources de lumière d'une scène. Il génère ensuite une image de cette scène à partir du point de vue d'une caméra elle aussi décrite dans le script.

La scène est décrite à l'aide d'un langage de description propre à POV-Ray. C'est une partie de ce langage que vous allez apprendre au cours de ce TP.

Dans toute la suite vous serez amené à utiliser l'aide de POVRay en français. Parcourez cet énoncé et cliquez sur les liens qui se présentent. Quand vous aurez lu la page correspondante de l'aide revenez à cet énoncé.

Le repère universel de POV-Ray

POV-Ray utilise un repère à main gauche.

Attention ! Le repère (O, Ox, Oy, Oz) de POVRay est indirect au sens mathématique du terme.

A l'aide de ce repère on peut définir des points et des vecteurs.

La gestion des variables

Dans un script POVRay on peut définir des variables, modifier leur valeur. Des directives POVRay, analogues à celles du préprocesseur C, permettent de gérer les variables.

Les fichiers include

Dans le script de description de la scène il est possible d'inclure des fichiers à l'aide de la directive #include.
